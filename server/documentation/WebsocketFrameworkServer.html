Class: WebsocketFrameworkServer

WebsocketFrameworkServer

A class that helps dealing with websocket communication between a server and clients.

Constructor

new WebsocketFrameworkServer(config)

Initiates the server-side Websocket Framework.
Parameters:
Name Type Description
config Object The configuration of the framework.
Properties
Name Type Attributes Default Description
maxClients Integer <optional>
 infinite The maximum amount of clients connected simultaneously to the server.
port Integer The port that will be used by the server to communicate.
timeoutDelay Integer <optional>
 10000 The delay after which a client will timeout if the server hasn't received a ping packet, in milliseconds.
onConnectionAttempt function <optional>
 A function that will be called when a client tries to connect (params: user). Can return an object: {successful: true/false, reason: ""}.
onConnection function <optional>
 A function that will be called when the connection with a client has been opened and authorized (params: user).
onConnectionClosed function <optional>
 A function that will be called when the connection with a client has been closed (params: user, data).
onPacket function <optional>
 A function that will be called when the server receives a packet from a client (params: type, data).
